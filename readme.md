# Proyecto probando Reflex con google Auth

## Importante para reflex
- Crear el entorno virtual  
  ```py -3 -m venv .venv```
- Arrancar el entorno virtual  
```.venv\Scripts\activate```
- Instalar dependencias  
```pip install -r requirements.txt```  
- Arrancar reflex  
```reflex run```
